Toys's API Documentation

  • Get 10 products by page number: toys/?page=number(GET)
  • Get 10 product's page: (default:10) toys(GET)
  • Search products by name or information: toys/search/?s=name(GET)
  • Get products by Category-name: toys/cat/:category(GET)
  • Price range: toys/prices/?max=x&min=y(GET)

Sign Up: /users(POST)

  • User(required)
  • Email(required)
  • Password(required)
  • Role

Sign In: /users/login(POST)

  • Email(required)
  • Password(required)

Add Product-(only for register user with token) toys(POST)

  • Name(required)
  • Price(required)
  • category(required)
  • Info
  • Img (URL)

Edit Product-(only for register user with token) /toys/:editid(PUT)

  • Name(required)
  • Price(required)
  • Cat(required)
  • Info
  • Img (URL)

Delete Product-(only for register user with token) /toys/:delid(Delete)
